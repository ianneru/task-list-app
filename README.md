# Task List App

## Tecnologias

>1. Angular 4.
>2. Bootstrap.

## Features

>1. Listagem de Tarefas.
>2. Adicionar Tarefas.
>3. Editar Tarefas.
>4. Excluir Tarefas.

## Como utilizar

>1. Rodar `npm install`.
>2. Rodar `ng serve` para servidor de desenvolvimento. Navegar para `http://localhost:4200/`.
